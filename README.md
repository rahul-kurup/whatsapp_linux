# Whatsapp (Linux)

A minimal WhatsApp Web wrapper for Linux, since no native client is available.

## Credits

- Major codebase: https://github.com/uvindusl/Whatsapp-for-linux (forked)
- Useragent Feed: https://jnrbsn.github.io/user-agents/user-agents.json

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
$ pnpm build
```
